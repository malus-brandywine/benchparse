# Parse sel4bench output

```
usage: python3 benchparse.py [-h] -b  -i  -o

Parse sel4bench output

optional arguments:
  -h, --help            show this help message and exit
  -b , --bench-output   File containing output of sel4bench
  -i , --benchmarks     File with a list of line-separated benchmarks
  -o , --benchparse-output 
                        File to write benchparse output to
```
